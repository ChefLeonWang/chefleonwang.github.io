Trust Region Policy Optimization (TRPO) is a reinforcement learning algorithm designed to ensure stable and monotonic policy improvement. It does so by solving a constrained optimization problem that explicitly limits how much the policy is allowed to change between updates.

üß† The Hessian Matrix

The Hessian matrix is the matrix of second-order partial derivatives of a scalar-valued function.

Given a function f:Rn‚ÜíRf: \mathbb{R}^n \rightarrow \mathbb{R}f:Rn‚ÜíR, the Hessian H‚ààRn√ónH \in \mathbb{R}^{n \times n}H‚ààRn√ón is defined as:

It captures how the gradient changes ‚Äî i.e., the curvature of the function surface.

  • If HHH is positive definite, the function is convex locally.
  • If HHH is negative definite, the function is concave locally.

In optimization problems like TRPO, the Hessian is approximated by the Fisher Information Matrix to reduce computational cost and improve stability.
